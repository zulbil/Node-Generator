# Node Generator
=====================

<p>
    It's a cli app that can generate a model schema only by providing some parameters 
</p>



### Installation

- git clone https://github.com/zulbil/Node-Generator
- npm install

### In Development
- npm run generate model : This command will help you generate a new model
